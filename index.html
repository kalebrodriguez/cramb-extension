Remember what you read.

Turn what you read into memory.

Cramb is a privacy-first, open-source browser extension that turns articles and YouTube videos into spaced-repetition flashcards — so the things you learn actually stick.

Add to Chrome Coming to the Chrome Web Store View on GitHub

How it works

1 · Capture

Highlight or one-click

Select text, capture an article, or grab a YouTube video's transcript — in seconds, without leaving the page.

2 · Generate

Editable cards

An AI model — your own API key or a local Ollama model — drafts flashcards. You review and edit before anything is saved.

3 · Review

Right on schedule

Cards are scheduled with the FSRS algorithm so you review at the optimal moment — keyboard-first, right in your browser.

Private by design

  • Local-first. Your sources and cards live in your browser (IndexedDB) — there's no Cramb account and no backend.
  • Bring your own key. Cramb never ships or shares an API key. It talks only to the model provider you choose.
  • No tracking. No analytics, no telemetry, no phone-home. Your reading stays yours.
  • Open source. Every line is on GitHub under the MIT license — audit it, fork it, contribute.

Install

Cramb isn't on the Chrome Web Store or Firefox Add-ons yet. Until then, you can build it from source and load it as an unpacked extension: 

git clone https://github.com/kalebrodriguez/cramb-extension
cd cramb-extension
pnpm install
pnpm build        # produces .output/chrome-mv3

Then open chrome://extensions, enable Developer mode, click Load unpacked, and select the .output/chrome-mv3 folder.
